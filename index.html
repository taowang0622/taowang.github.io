A Seckill Backend System

Frontend: Bootstrap, AngularJS

Backend: Spring Framework, MyBatis, MySQL, Redis, AWS EC2 and RDS, Docker

A backend system is to handle the situation where many people purchase one item at the same time which is highly concurrent situation.

  • In this system, one client is only allowed to purchase one product once.
  • To reduce the effect of network latency between JAVA client and MySQL and GC, I use Redis to cache data retrieved from MySQL.
  • Since it happens a lot in this system that many threads update the same row in MySQL at the same time, it's necessary to reduce the time of each thread holding row-level locks. I use SQL stored procedures and restructure the transaction logic to achieve this.

A Simple Shopping Website

Spring Framework, MyBatis, MySQL, Freemarker, Docker, AWS....

A simple shopping website that serves two types of clients: buyer and seller.

  • As a buyer, you can add wanted items into your shopping cart and purchase them all at once and check your purchase history in the account page.
  • As a seller, you can add new items to your website, and delete them anytime, also you can edit them anytime
